# VOICEVOX Engine

このディレクトリにVOICEVOX Engineを配置してください。

## ダウンロード方法

1. 以下のURLから最新版をダウンロード:
   https://github.com/VOICEVOX/voicevox_engine/releases

2. Windows CPU版を推奨:
   voicevox_engine-windows-cpu-0.25.0.7z.001

3. 解凍してこのディレクトリに配置

## 必要なファイル構造

voicevox-engine/
├── run.exe (またはrun.bat)
├── (その他のエンジンファイル)

## 注意

.gitignoreに追加されているため、エンジン本体はGitリポジトリに含まれません。
ビルド時には別途エンジンを配置する必要があります。
